# 🌿 EcoTrack AI — Sustainable Future by Parth Valand

EcoTrack AI is a powerful, AI-driven carbon footprint analysis system designed to help users track, analyze, and reduce their daily environmental impact.

## ✨ Features
- **🤖 AI Eco-Bot:** Personalized sustainability advice powered by Google Gemini AI.
- **📊 Interactive Dashboard:** Data visualization using Recharts (Line, Bar, Pie).
- **📝 Smart Calculator:** Real-time carbon footprint estimation based on Transport, Electricity, Fuel, and Diet.
- **🏆 Gamification:** Leaderboard, badges, and sustainability quizzes.
- **📄 Export Options:** Professional PDF reports and TXT data history.
- **📱 PWA Support:** Installable as a desktop or mobile application.
- **🌍 Multi-language:** Supports English, Hindi, and Gujarati.
- **🌙 Dark/Light Mode:** Premium glassmorphism UI with theme switching.

## 🛠️ Tech Stack
- **Frontend:** React, Vite, Framer Motion, Axios, Recharts.
- **Backend:** Python, Flask, JWT Authentication.
- **Database:** SQLite.
- **AI Integration:** Google Gemini Pro API.

## 🚀 How to Run Locally

### 1. Clone the repository
```bash
git clone <YOUR_GITHUB_REPO_LINK>
cd EcoTrack-AI
```

### 2. Setup Backend
```bash
cd backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

### 3. Setup Frontend
```bash
cd frontend
npm install
npm run dev
```
