**EcoTrack AI** is a state-of-the-art, full-stack sustainability platform. It empowers users to monitor their carbon footprint, receive AI-driven advice, and compete in a global leaderboard to promote a greener lifestyle.

---

## ✨ Core Features

- **🤖 AI Eco-Bot:** Personalized sustainability coaching powered by **Google Gemini AI**.
- **📊 Advanced Analytics:** Interactive Line, Bar, and Pie charts to track monthly emissions.
- **📝 Precision Calculator:** Estimate emissions from Transport, Electricity, Fuel, and Diet.
- **🏆 Gamification:** Earn badges, collect points, and climb the **Global Leaderboard**.
- **📄 Pro Reporting:** Download professional **PDF Certificates** and TXT data exports.
- **📱 PWA Ready:** Install it as a standalone app on your Mobile or Desktop.
- **🌍 Multi-lingual:** Full support for English, Hindi, and Gujarati.
- **🌙 Glassmorphism UI:** Ultra-modern dark/light mode with smooth Framer Motion animations.

## 🛠️ Architecture

- **Frontend:** React 18, Vite, Framer Motion, Axios, Recharts.
- **Backend:** Python 3, Flask, JWT Auth, Scikit-learn (ML).
- **Database:** SQLite (Relational).

## 🚀 Installation & Setup

### 1. Clone & Install
```bash
git clone https://github.com/Parthvaland2005/EcoTrack-AI.git
cd EcoTrack-AI
```

### 2. Run Backend
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### 3. Run Frontend
```bash
cd frontend
npm install
npm run dev
```
